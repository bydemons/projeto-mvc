## 🧱 Estrutura do Projeto (MVC)

- models/ → Responsável pela conexão com o banco de dados e regras de negócio  
- views/ → Responsável pela interface do usuário (HTML)  
- views/css/ → Arquivos de estilo (CSS)  
- views/js/ → Scripts do lado do cliente (JavaScript)  
- controllers/ → Responsável pela lógica da aplicação e controle de rotas
