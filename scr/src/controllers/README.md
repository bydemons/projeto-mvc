# Projeto Web - MVC

## 🧱 Estrutura do Projeto

- models/ → conexão com banco, regras de negócio  
- views/ → HTML + interface  
- views/css/ → estilos  
- views/js/ → scripts do front  
- controllers/ → lógica (rotas, ações do usuário)

---

## 📌 Menu Principal

- Início
- Login
- Cadastro
- Perfil

---

## 🔄 Fluxo do Usuário

1. Acessa a página inicial  
2. Vai para cadastro ou login  
3. Entra no sistema  
4. Usa a funcionalidade principal  

---

## ♿ Acessibilidade

- Botões com cores contrastantes  
- Textos simples e claros  
- Navegação fácil
